Detailed Description

Data dimension struct.

Used as meta information of array data in gpuNUFFT::Array.

Stores array length (1-d), width, height and depth (3-d), channels (coils, 4-d) and frame (5-d) dimensions.

Default values: width = 0, height = 0, depth = 0, channels = 1, frames = 1, length = 0

Warning: currently no distinction between one-dimensional (length) and higher-dimensional (width, height, depth) is made, thus ambiguity can occur in count() computation.
