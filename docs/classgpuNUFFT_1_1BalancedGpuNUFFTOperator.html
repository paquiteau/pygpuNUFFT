Detailed Description

Balanced GpuNUFFTOperator inherited from gpuNUFFT::GpuNUFFTOperator.

Changes the behaviour of the default GpuNUFFTOperator by balancing the work load by sector to a maximum amount of samples per sector (MAXIMUM_PAYLOAD). Thus, sectors with a high density of data points are split into multiple ones, which are processed in parallel.
