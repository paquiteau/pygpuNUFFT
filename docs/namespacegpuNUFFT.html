enum gpuNUFFT::InterpolationType

Type used for texture interpolation on GPU.

The kernel used for gridding interpolation is loaded in either a const array or in a 1-d, 2-d or 3-d texture.

Enumerator

No texture lookup is performed.

TEXTURE_LOOKUP
