Detailed Description

Definition of util functions used in gridding operations.

The functions are based on the grid3 work by Nick Zwart

Lookup table creation extracted from GRID_UTILS.C

Author: Nick Zwart, Dallas Turley, Ken Johnson, Jim Pipe Date: 2011 apr 11 Rev: 2011 aug 21 In: grid3_dct_11aug
