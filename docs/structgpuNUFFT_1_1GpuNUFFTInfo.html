Detailed Description

Struct containing meta information of the current Gridding Problem. Used in most GPU operations.

Stores neccessary information required in most of the GPU gridding and auxiliary processing steps.

The GpuNUFFTInfo is loaded to a constant variable (GI) in the GPU context.

Initialization is performed in gpuNUFFT::GpuNUFFTOperator::initGpuNUFFTInfo() function.
